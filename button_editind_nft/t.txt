Para criar um efeito de carregamento de elementos com animações em React, você pode usar bibliotecas como react-transition-group ou simplesmente usar CSS para definir suas animações personalizadas. Aqui está um exemplo de como você pode criar uma animação de entrada da esquerda para a posição do elemento:Primeiro, certifique-se de que você instalou o react-transition-group:npm install react-transition-groupEm seguida, importe a biblioteca no seu componente React:




import { TransitionGroup, CSSTransition } from 'react-transition-group';Crie um componente que renderizará os elementos animados:import React, { useState } from 'react';
import './App.css'; // Importe seu arquivo CSS para estilização

function App() {
  const [items, setItems] = useState([]); // Estado para armazenar os itens a serem animados

  const addItem = () => {
    const newItem = `Item ${items.length + 1}`;
    setItems([...items, newItem]);
  };

  return (
    <div className="App">
      <button onClick={addItem}>Adicionar Item</button>
      <TransitionGroup>
        {items.map((item, index) => (
          <CSSTransition key={index} classNames="item" timeout={500}>
            <div className="item">{item}</div>
          </CSSTransition>
        ))}
      </TransitionGroup>
    </div>
  );
}

export default App;


Em seu arquivo CSS (App.css neste exemplo), defina as animações:/* Adicione uma classe de entrada personalizada */
.item-enter {
  transform: translateX(-100%); /* Comece fora da tela à esquerda */
}

/* Defina a transição para suavizar a animação */
.item-enter-active {
  transform: translateX(0); /* Traga o elemento para a posição original */
  transition: transform 500ms;
}

/* Defina a saída se você desejar uma animação de saída ao remover elementos */
.item-exit {
  transform: translateX(0); /* Comece na posição original */
}

/* Defina a transição de saída */
.item-exit-active {
  transform: translateX(100%); /* Mova o elemento para a direita e fora da tela */
  transition: transform 500ms;
}Com essas etapas, você terá uma animação de entrada da esquerda para a posição do elemento quando você adicioná-lo à lista. Você pode ajustar as classes CSS e as propriedades de animação conforme necessário para atender às suas necessidades específicas.
